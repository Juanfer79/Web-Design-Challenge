Summary: Latitude Comparisons

Wellcome to the geolocation weather analytics proect. In here we will review the correlation between the Latitude of 500 Cities randomly selected from OpenWeatherMap API.

Matplotlib was the tool within Python used to plot the figures for the 500 cities and be able to analyze diferent weather aspects as Tempeture, claudiness, Windspeed and humidity. Several plots were created to present results and inhere you will be able to read the analysis for each anlytics graph
